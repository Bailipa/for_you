## 需求说明

### 1. 空状态

- 没有会话时，侧栏显示"暂无会话"的提示
- 当前会话没有消息时，聊天区域显示欢迎页面：
  - 应用名称（如"小豆"）
  - 简短介绍文字
  - 几个示例问题（点击可自动填入输入框并发送）

### 2. 自动滚动

- 新消息出现时自动滚动到底部
- 如果用户向上滚动查看历史消息，不自动滚动
- 用户手动滚动到底部后恢复自动滚动
- AI 流式回复时持续跟随最新内容

### 3. 加载与动画

- AI 回复时的打字机效果（逐字出现）
- 消息出现时的淡入动画 (Tailwind animate)
- 侧栏展开/收起时的平滑过渡 (transition)

### 4. 错误处理

- API 调用失败时在聊天区域显示红色错误提示
- 提供"重新发送"按钮
- 错误信息要友好（如"网络连接失败，请检查 API 配置后重试"）

### 5. 快捷键

- Enter 发送，Shift+Enter 换行
- 输入框为空时发送按钮禁用

### 6. 为项目写 README

在项目根目录创建 `README.md`，内容包括：

```markdown
# 小豆 - AI 对话助手

一个仿"豆包"风格的 AI 聊天应用。

## 功能

- AI 智能对话
- 多会话管理
- 消息历史持久化
- 流式输出（打字机效果）
- 响应式设计

## 技术栈

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI API

## 快速开始

1. 复制 `.env.local.example` 为 `.env.local` 并填入 API Key
2. `npm install`
3. `npm run dev`
4. 打开 http://localhost:3000
```
